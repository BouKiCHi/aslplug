## gdbメモ
現在、最新版のgdbに不具合があり、動作しません。

https://github.com/Alexpux/MINGW-packages/issues/1862

MSYS2環境で以前のバージョンのgdbを直接インストールする方法として
pacman -U http://repo.msys2.org/mingw/i686/mingw-w64-i686-gdb-7.11.1-4-any.pkg.tar.xz
があります。
