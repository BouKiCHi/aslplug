nezplay_asl

nezplug++をベースにした、
KSS再生時のNLG/S98V3出力機能を備えるコマンドラインプレイヤーです。

■MDR再生について
MDRファイルを再生するには、MDRファイルと同じディレクトリ、またはワーキングディレクトリに
moon.binを配置してください。

■ドライバのファイルパス
検索順番は、
実行ファイルと同じディレクトリ>曲ファイルと同じディレクトリ>カレントディレクトリ
となります。

512バイト以上のサイズでドライバとみなします。
