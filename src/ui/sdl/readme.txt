nezplay_asl

nezplug++をベースにした、
KSS再生時のNLG/S98V3出力機能を備えるコマンドラインプレイヤーです。

■MDR再生について
MDRファイルを再生するには、MDRファイルと同じディレクトリ、またはワーキングディレクトリに
moon.binを配置してください。

■ドライバのファイルパス

ディレクトリの検索順番は、
曲ファイルと同じ > 実行ファイルと同じ > ドライブ検索パス > カレントディレクトリ
となります。

512バイト以上のサイズでドライバとみなします。

■Tips
曲時間(-l/--lenオプション)は秒数だけではなく、2:15(=135秒)などと指定できます。

■リアルタイム出力モード
c86ctl.dllを実行ファイルと同じディレクトリに置いて実行してください。

■オプション解説
＃いくつかのオプションは無効な可能性があります。


-s / --rate <rate>   : 再生周波数の設定
-v / --vol <vol>     : 音量の調整(1.0が標準)
-l / --len <n|mm:ss>       : 曲の長さを秒数で設定 (n secs)

--rt         : 実デバイス用リアルタイム出力モード

-n <no>      : 曲番号の設定
-q <dir>     : ドライバパスの設定

-o <file>    : WAVファイルの出力
-p           : NULL PCM モード(無音出力)

-9 / --s98   : S98V3 ログモード
-u / --rough : S98 ラフテンポモード (default)
--fine       : S98 ファインテンポモード
-r <file>    : サウンドログを記録する
-b           : 曲ファイル名を元にサウンドログを記録する(無音)
-g          : OPMにFMGENを使用する
-a          : 高速CPUモードを強制する
-x          : 厳密モード
-w          : 詳細モード

-h / --help : ヘルプ
