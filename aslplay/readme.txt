
ASLPLAY

■説明
　このプログラムは、nezplug++をベースにした、
KSS再生時のNLG/S98V3出力機能を備えるコマンドラインプレイヤーです。

　NRTDRVやMoonDriverなどのデータを再生することができます。


■MDR再生について

　MDRファイルを再生するには、MDRファイルと同じディレクトリ、
またはワーキングディレクトリにmoon.binを配置してください。


■ドライバのファイルパス


　ディレクトリの検索順番は、
曲ファイルと同じ > 実行ファイルと同じ > ドライブ検索パス > カレントディレクトリ
となります。

現在は512バイト以上のファイルサイズでドライバが存在するとみなします。


■Tips


　曲時間(-l/--lenオプション)は秒数だけではなく、2:15(=135秒)などと指定できます。


■リアルタイム出力モード


　MoonDriverまたはNRTDRVの楽曲再生時に実デバイスに出力するモードです。
c86ctl.dllを実行ファイルと同じディレクトリに置いて実行してください。


■オプション解説


＃いくつかのオプションは無効として扱われる可能性があります。

-s / --rate <rate>   : 再生周波数の設定
-v / --vol <vol>     : 音量の調整(1.0が標準)
-l / --len <n|mm:ss> : 曲の長さを秒数で設定 (n secs)

--rt         : 実チップ用出力モード

-n <no>      : 曲番号の設定
-q <dir>     : ドライバパスの設定

--wav        : WAVファイルの出力
-o <file>    : WAVファイルの出力(ファイル名指定)
-p           : NULL PCM モード(無音出力)
--nosound    : NULL PCM モード(無音出力)

--dump       : ダンプモード

--log        : サウンドログを記録する
-r <file>    : サウンドログを記録する(ファイル名指定)
-b           : 曲ファイル名を元にサウンドログを記録する(無音)
--nlg        : NLG ログモード
-9 / --s98   : S98V3 ログモード(default)
-u / --rough : S98 ラフテンポモード (default)
--fine       : S98 ファインテンポモード

--title <str>   : S98タグ タイトル
--artist <str>  : S98タグ　アーティスト
--game <str>    : S98タグ　ゲーム

--sub <file>   : サブスロットの曲ファイルを指定
--subvol <vol> : サブスロットの音量を指定(def 1.0)
--subnum <num> : サブスロットの曲番号を指定

-g          : OPMにFMGENを使用する
-a          : 高速CPUモードを強制する
-x          : 厳密モード
-w          : 詳細モード

-h / --help : ヘルプ
